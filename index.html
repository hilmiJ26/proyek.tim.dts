Apa yang Akan Kamu Pelajari di CODEACADEMY?

Fundamental HTML, CSS, JS

Anda belum punya basic coding sama sekali sebelumnya ? Jangan kuatir, Anda akan diperkuat terlebih dahulu materi-materi seputar fundamental html, css, es6, hingga Javascript itu sendiri.

React, Router, Redux

Anda akan diajak praktek bagaimana untuk membangun sebuah tampilan frontend yang ciamik menggunakan framework React, Router, Redux akan diberikan bagaimana penerapan-penerapannya.

Building Real Apps

Rasakan pengalaman benar-benar membangun aplikasi real dengan berbagai fitur mulai dari login, registrasi, checkout, add to cart, pembayaran, step by step.

NodeJS, REST API, Design Pattern

Anda diberikan pemahaman kuat membangun Backend aplikasi menggunakan NodeJS. Rest API, Database, dan design pattern juga akan diperdalam.
